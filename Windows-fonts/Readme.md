# 中文使用说明：

- ① 下载或提取所需的`Windows`字体文件  
  （可`ttf`文件，可`ttc`文件，可`woff`/`woff2`文件），

- ② 与此脚本文件（`fonts-installer.sh`）放在同一文件夹下，

- ③ 使此脚本文件处于可执行状态  
  （终端下运行`sudo chmod a+x fonts-installer.sh`命令  
  或  
  右键“属性”中的“权限”栏点击“允许此文件作为程序运行”），

- ④ 执行此脚本文件(终端下运行`./fonts-installer.sh`命令或双击此文件)，

- ⑤ 即可获得与`Windows`系统相同的字体文件。

> 请尽情享受吧！

# English

- ① Download or extract the required `Windows` font file
  (can be `ttf` files, `ttc` files, or `woff`/`woff2` files),

- ② It is placed in the same folder as this script file (` fonts installer. sh `),

- ③ Make this script file executable
  (Run the `sudo chmod a+x fonts installer. sh` command under the terminal
  Or
  Right click the "Permission" column in the "Properties" and click "Allow this file to run as a program"),

- ④ Execute the script file (run the `./fonts installer. sh` command on the terminal or double-click the file),

- ⑤ You can get the same font file as the Windows system.

> Please enjoy yourself!
